## 使用教學

### 一、安裝前準備

1. 安裝 SteamTools
   - 前往 [3a.lol](https://3a.lol) 下載並安裝 SteamTools
   - 安裝完成後，確保 SteamTools 正常運作

2. 修復 Steam
   - 使用 SteamTools 的修復功能修復你的 Steam 客戶端
   - 確保 Steam 能夠正常登入和運作

### 二、安裝 FluentInstall

1. 下載程式
   - 從上方下載地址下載最新版本的 FluentInstall
   - 將壓縮包解壓到任意資料夾

2. 安裝依賴
   - 執行 fluent_install.bat 安裝必要的依賴
   - 等待安裝完成

3. 啟動程式
   - 執行 python main.py 啟動程式
   - 或者執行 run_fluent.bat 快速啟動

### 三、使用程式

#### 1. 搜尋並入庫遊戲

1. 點擊左側導航欄的「搜尋入庫」
2. 在搜尋框中輸入遊戲名稱或 AppID
3. 點擊「搜尋」按鈕
4. 從搜尋結果中選擇你要入庫的遊戲
5. 選擇清單源（如果某個源失敗，請嘗試其他源）
6. 根據需要勾選入庫選項：
   - 添加所有 DLC：同時入庫遊戲的所有 DLC
   - 創意工坊入庫：入庫遊戲的創意工坊內容
   - 修補 Manifest（不建議選，不能用再選）：修復清單檔案
7. 點擊「入庫遊戲」按鈕

#### 2. 管理已入庫遊戲

1. 點擊左側導航欄的「主頁」
2. 查看所有已入庫的遊戲列表
3. 可以切換檢視模式（列表/卡片）和排序方式
4. 點擊遊戲卡片上的「刪除」按鈕可以移除已入庫的遊戲

#### 3. 重啟 Steam

1. 入庫完成後，點擊右上角的「重啟 Steam」按鈕
2. 確認重啟後，Steam 將自動關閉並重新啟動
3. 重啟後，入庫的遊戲將出現在你的庫中

### 四、設定

1. 點擊左側導航欄的「設定」
2. 可以配置以下選項：
   - Steam 路徑：手動指定 Steam 安裝路徑（留空則自動檢測）
   - 主題模式：淺色/深色/跟隨系統
   - 主題顏色：選擇你喜歡的主題顏色
   - 語言：切換介面語言
   - 預設頁面：設定啟動後顯示的預設頁面
   - 解鎖工具模式：自動檢測/強制 SteamTools/強制 GreenLuma
   - 入庫超時時間：網路較慢時可適當增大

### 五、注意事項

1. 確保 SteamTools 已安裝並正常運作
2. 入庫前建議先重啟 SteamTools
3. 如果入庫失敗，嘗試更換清單源
4. 某些遊戲可能需要特定的清單源才能入庫成功
5. 入庫後必須重啟 Steam 才能看到遊戲
